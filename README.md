# Dashboard Project
Este é um projeto de painel administrativo desenvolvido para demonstrar habilidades em estruturação de layout, manipulação de dados via JavaScript e design responsivo.
> [!TIP]
> Este dashboard foi construído pensando em performance e fidelidade visual, sem a necessidade de frameworks pesados.
## Tecnologias Utilizadas

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização avançada e layouts flexíveis.
- **JavaScript:** Lógica de interatividade e manipulação de dados dinâmicos.
[1]
## Funcionalidades
- [x] Interface responsiva para dispositivos móveis
- [x] Organização modular de arquivos (CSS/JS/Data)
- [ ] Integração dinâmica com gráficos
- [ ] Modo Escuro (Dark Mode)
## Estrutura de Pastas
| Diretório | Descrição |
| --- | --- |
| /css | Arquivos de estilização e temas |
| /js | Scripts de comportamento e lógica |
| /data | Arquivos de dados ou JSON de exemplo |
## Como Executar
Para visualizar o projeto localmente, basta clonar o repositório e abrir o arquivo *index.html* no seu navegador:
```bash

git clone https://github.com/RicardoFiorini/dashboard.git
cd dashboard[1]
Abra o index.html no navegador

```
